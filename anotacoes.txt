 npm init -y

npm i bcryptjs cors dotenv express express-validator jsonwebtoken mongoose multer
- bcryptjs          - permite a gente ter uma melhor manipulação de senhas com javascript;
- dotenv            - variaveis de ambiente de nossa aplicacao (dominio, banco de dados);
- express           - framework backend para criacao de apis;
- express-validator - é como se fosse um middleware, vai trabalhar entre as requisicoes para validar os dados (MVC);
- jsonwebtoken      - gerar e verificar os tokens para a gente autenticar o usuario ou deixar o token ativo;
- mongoose          - a ferramenta que vai utilizar para trabalhar com o banco de dados;
- multer            - é uma ferramenta que a gente vai utilizar para trabalhar com o upload de fotos ou imagens;

npm i --save-dev nodemon
- dependencia de dev que simula um servidor local que tem uma funcao de carregamento de codigo automatico
